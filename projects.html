Мои проекты

AoLanguage

Плагин для PocketMine-MP, обеспечивающий мультиязычную поддержку на серверах Minecraft. Позволяет игрокам выбирать язык интерфейса, упрощая взаимодействие для международного сообщества. Написан на PHP с акцентом на производительность и простоту интеграции.

Download #pocketmine-pm5

Seasons

Плагин для PocketMine-MP, добавляющий динамическую систему сезонов в игровой мир. Включает изменения погоды, текстур и событий, зависящих от времени года, для повышения вовлечённости игроков. Разработан с использованием PHP и API сервера.

Quests

Плагин для PocketMine-MP, реализующий систему квестов. Игроки могут выполнять задания, получать награды и участвовать в сюжетных линиях. Разработан для гибкости, с поддержкой кастомизации через конфигурационные файлы.
